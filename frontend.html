Front-end Challenge

Part 1: Map

Given a set of geographic data about some of our patients, create an HTML page with D3 containing a graph of the U.S. The dataset is available as a CSV file here.

The CSV has columns City, State, Zip, Patient ID, and pharmacy. Each row represents a single patient. The graph should show show each state in the US, and the color of the state should vary depending on the number of patients in that state (states with more patients, should be darker than states with fewer patients).

You may use raw D3 or D3 wrapper libraries.

Tip: You will need to aggregate patients from the same state. You can do this 'live' on the page, or pre-process the CSV on own before rendering.

Part 2: Interactions

Once the graph is rendered, add the following 2 interactions:

  1. When hovering over a state, change the color of the state to black
  2. When clicking on a state, show a JavaScript alert with the selected state's name and patient count

Rules

Please use any external libraries or code snippets that you see fit, but please include links / references to whatever you use in your submission.

Submission

When finished, zip up all the files required to load your page, and email them to swaraj@semprehealth.com.

Extra credit for:

  • Hosting the page at a public URL
  • Adding additional, custom interactions and animations
